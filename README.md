# Smartphone Behavior-Based Intrusion Detection System

## Overview
This project is an AI-based system that detects abnormal smartphone behavior to identify potential intrusions and security threats.

## Features
- Continuous behavior monitoring
- Anomaly detection using AI
- Real-time alert system
- Secure authentication mechanism

## Technologies Used

- Machine Learning
- FastAPI (optional)
- Frontend Interface
